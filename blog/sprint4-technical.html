Time when I got blocked on a simple problem


I was trying to find a setup file to run for software but I could not find where the file was so I spent hours finding a way on how to install the software. I looked up on Google and also looked for a readme file in the folder to see instructions but there was nothing anywhere. I was frustrated and then I checked with the support team and realized that all I had to do was copy and paste and there was no need to install it on the computer.



Problem solved in an elegant way.
